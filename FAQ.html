How Do I Install Themes?

For BetterDiscord My Themes Can be located on the BetterDiscord's Site , simply click download on the theme you wish to download and then place this file in your BetterDiscord's theme directory.

How To Edit The Theme?

Within The Chosen Application You Chose, BetterDiscord Or Powercord Locate The File .theme For BD & the File In The Powercord Manifest To Edit / Change The Theme. Its Good Practise To Change This In Quick CSS Though To Keep Automatic Updates Rolling

To Toggle The Colours Change The #00000; Variables These Are HEX Codes, Some Are RBG (100, 100, 100, 0.25) and what else, Use an editor like visual studio to make this easier or an online editor.

How to get future updates?

For BetterDiscord Users You Will Need To Manually Redownload Your Theme From BetterDiscord's Site . If you are on Powercord you will have automatic updates with the updater plugin already installed.
