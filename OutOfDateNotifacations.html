SO.. YOUR INTERESTED CREATING A OUTDATED NOTIFACTION?

Below I'll Teach A Way On How To Automatically Tell Your Users How To Updated Since There Theme Is Either Broken OR Outdated And Need To Update!

STEP ONE

First You Will Need A Website Already Made Which Is Hosting Your Files Meaning it can check for updates. I Have Made A Guide Here.

Once On This Website We Need To Have A Custom Folder For Each Theme We Like To Auto Update.

And One For Global Changes To The Banner If You Like But I Prefer Custom Per Theme.

Need Help

STEP TWO

Now In  The Folder create a file called " version " We Put This:
The Top Section Of https://leeprky.github.io/themes/Bloop/version.css

Now We Need To Create The Banner Now So:

We Create A Variable Called (--outdated-theme-[THEME-BUILD],block)
and everytime we update the theme 
WE MUST CHANGE THESE TO MATCH THE LATEST BUILD OTHERWISE YOU WILL HAVE INCORRECT DISPLAYING OF THE BANNER. 

COPY THIS TOO AND CHANGE THE DETAILS

#app-mount > div.app-1q1i1E > div > div.layers-3iHuyZ.layers-3q14ss > div > div > div.base-3dtUhz:before{display:var(--outdated_[THEMENAME-[BUILD#],block)!important;position:absolute;left:10%!important;width:fit-content;border-radius:0 0 12px 12px!important;top:00px!important}

Lastly We Need The Actual Banner To Display:















SHOULD BE LOOKING LIKE BELOW

--theme= The Themes Name

--build-latest = The Commit Number or update number like 001 002... whatever u like just make it going up chronologically.

--version-latest = The Latest Version Of The Theme

--version-latest-long = The Latest Version In Long Text
 Version: V1.1.0 (Latest)

--version-long = The Themes Current Locally Installed Version
Version: V1.0.0 (Current)

--theme-dl = The Themes URL

STEP THREE

WE ARE NOW INSIDE THE THEMES-CONFIG.CSS OR .THEME FILE

At The Top Of The Theme File We Need  The Url Of The Code We Just Made In STEP TWO:

@import "https://leeprky.github.io/themes/Bloop/version.css";

And Now At The Very Bottom We Need:

:root {--outdated_bloop-[THEME-BUILD]: none !important; 

--build: "[THEME-BUILD]" !important}

The --outdated_bloop--[THEME-BUILD] should be changed to match the one on the server so that if this in the local file is different it will display the banner if its the same it wont display the banner.
